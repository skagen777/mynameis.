MY NAME IS. V8.3 STABLE

이번 버전은 PC와 모바일 CSS를 완전히 새로 분리했습니다.
기존 style.css / script.js를 사용하지 않습니다.

새 파일:
- site-v83.css
- site-v83.js

중요 적용 방법:
1. GitHub 저장소의 기존 파일을 전부 삭제합니다.
2. ZIP을 압축 해제합니다.
3. assets 폴더를 포함한 ZIP 내부 전체를 업로드합니다.
4. Commit changes 후 브라우저에서 강력 새로고침합니다.
   - Windows: Ctrl + Shift + R
   - Mac: Command + Shift + R

레이아웃:
- PC: 한 화면 3×3 모자이크
- 모바일: 중앙 로고 전체 폭 + 2열 갤러리
- 모바일 첫 탭 제품 미리보기, 두 번째 탭 상세 이동
- 상세 사진은 모바일 가로 스와이프
