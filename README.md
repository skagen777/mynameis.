# MY NAME IS. Website V3

## 업로드 방법
1. 압축을 풉니다.
2. 저장소의 `Add file → Upload files`로 이동합니다.
3. 파일 전체와 `assets` 폴더를 함께 업로드합니다.
4. `Commit changes`를 누릅니다.

## 사이트 주소
https://my-nameis.com

## 핵심 구조
- 장소가 화자입니다.
- 향은 장소의 목소리를 번역한 결과입니다.
- `Meet / Archive / Dispenser / Who is speaking?` 구조입니다.

## 이미지 교체
`assets` 폴더 안의 SVG 파일을 실제 사진으로 바꿀 수 있습니다.
파일명만 동일하게 유지하면 HTML 수정 없이 교체됩니다.
